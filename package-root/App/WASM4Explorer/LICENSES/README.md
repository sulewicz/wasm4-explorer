# License Files

This directory is included in the Explorer app package so license and attribution material travels with the installed add-on.

Included now:

- `project_LICENSE.txt`: GNU GPL version 3.0 license text for repository-owned project code, scripts, docs, and packaging metadata.
- `wasm4_libretro_LICENSE.txt`: upstream WASM-4 license text for the bundled libretro runtime.
- `wasm4_gallery_license.txt`: CC BY-NC-SA 4.0 license text for WASM-4 gallery cart metadata and downloaded gallery carts.
